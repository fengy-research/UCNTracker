Dependency

  The package depends on
    GLIB 2.10+:
      http://ftp.gnome.org/pub/gnome/sources/glib/
      In general, any version newer than 2.10 should work.
    VALA 0.5.7:
      http://download.gnome.org/sources/vala/0.5/
      I have prebuilt packages on RHEL5 and Fedora 10.
    GSL 1.2:

Compiling
=========
  # these commands should be sufficient 
  $ ./autogen.sh
  $ make

Code Layout
===========
  UCNTracker       : git repository root
  |-- autotools    : macros and scripts used by autotools
  `-- ucntracker   : the main ucntracker library is built here
      |-- builder  : UCNBuilder, the geometry xml parser.
      |-- core     : the real ucntracker
      |-- library  : the library interface
      `-- tests    : very minimal test cases

  This layout is up to several changes, eg:
   (1) there is no place for plugin-modules(such as new geometry objects)
   (2) there is no place for bundled applications
